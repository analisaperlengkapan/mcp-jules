# Contributing

Thanks for helping improve Jules MCP Server.

## Setup

1. Install Node.js 18 or newer.
2. Install dependencies with `npm install`.
3. Build the project with `npm run build`.
4. Run the check script with `npm run check` before opening a pull request.

## Development flow

- Create a topic branch for your change.
- Keep changes focused and small where possible.
- Update `README.md`, `CHANGELOG.md`, or related docs when behavior changes.
- Include tests or validation steps when a change affects runtime behavior.

## Pull requests

- Describe the problem and the solution clearly.
- Link related issues when available.
- Mention any setup steps, API changes, or follow-up work.
- Confirm the project still builds locally.

## Reporting issues

- Include the command you ran, the expected result, and the actual result.
- Share relevant logs or stack traces when possible.
- For security issues, use `SECURITY.md` instead of opening a public issue.
